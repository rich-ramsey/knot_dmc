This project starts with plotting the knot dmc data as a summary plot.

It also creates a manuscript using the papaja() package.

The model in /models/ is taken from Sam's OSF page.

The figures in the /figures/from_file/ folder are also taken from Sam's OSF page.
This project is therefore a supplement to the main analyses performed by Sam, 
which creates one extra plot and makes the manuscript more reproducible via the use of papaja().
